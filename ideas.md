**Design Movement:** British Editorial Modernism — inspired by quality print magazines like Monocle and Wallpaper*

**Core Principles:**
- Authoritative yet approachable: a guide that feels trustworthy and professionally produced
- Strong typographic hierarchy drives all layout decisions
- Restrained colour palette with a single bold accent
- Content-first: information is the hero, not decoration

**Color Philosophy:**
- Deep forest green (#1a3a2a) as the primary background for the hero — evoking tennis courts and outdoor spaces
- Off-white (#f5f0e8) for body sections — warm, paper-like, not clinical
- Bright acid yellow (#d4e84a) as the single accent — energetic, modern, echoes tennis ball colour
- Charcoal (#1c1c1c) for body text

**Layout Paradigm:**
- Asymmetric two-column layout for main content sections
- Left-rail navigation that sticks on scroll for desktop
- Full-bleed hero with large display type offset to the left
- Tables styled as editorial data-tables with ruled lines, not card grids

**Signature Elements:**
- Large oversized section numerals (01, 02, 03) in a light weight as decorative anchors
- Thin horizontal rules between sections
- Pull-quote blocks in the accent yellow

**Interaction Philosophy:**
- Smooth scroll between sections via anchor links
- Hover states on links and cards use a subtle underline slide-in
- Section entrance animations: fade-up on scroll

**Animation:**
- Entrance: elements fade up with a 40px translate over 0.5s ease-out, staggered by 0.1s
- No looping or distracting animations; motion is purposeful

**Typography System:**
- Display: Playfair Display (serif) — bold, editorial weight for headings
- Body: DM Sans (sans-serif) — clean, readable for paragraphs and tables
- Mono: JetBrains Mono for contact details/URLs

- Single-column editorial layout with generous max-width (720px for text, 1100px for tables/cards)
- Sticky top navigation with the club's guide title

**Signature Elements:**
- Decorative tennis ball motif (SVG circle with curved lines) used sparingly as a section marker

- Anchor navigation highlights the active section
- Contact cards have a gentle hover lift with a green border reveal

- Sections fade in on scroll with a gentle 30px upward translate over 0.4s
- Navigation active indicator slides smoothly between items

**Typography System:**
- Display/Headings: Cormorant Garamond (elegant serif, multiple weights)
- Body: Source Sans 3 (highly legible, professional)

## Selected Approach: Refined British Club Aesthetic

Choosing **Option 3: Refined British Club Aesthetic** — warm cream backgrounds, deep Wimbledon green, and gold accents. Cormorant Garamond for headings, Source Sans 3 for body text. Single-column editorial layout with brand contact cards in a responsive grid. This feels authoritative, summery, and appropriate for a UK tennis club audience.
